============================================================
   Xbench FP Filter — 使用说明
============================================================

## 环境要求
- Windows 10/11
- 网络连接（首次运行需下载约 10MB Python 运行环境）

## 快速开始

1. 解压到任意目录
2. 双击 run.bat
3. 首次运行自动安装环境 → 浏览器打开 Streamlit
4. 在「参数设置」页输入 API Key 和 Base URL
5. 回到「文件处理」页上传 Xbench 报告 → 开始过滤

## 文件说明

├── run.bat         启动脚本（双击即可）
├── setup.bat       环境安装脚本（run.bat 会自动调用）
├── config_template.py  配置模板，参考抄到 config.py
├── requirements.txt    Python 依赖清单
└── README.md       开发者文档

## 常见问题

Q: 缺少 DLL / 无法启动 Python？
A: 安装 Visual C++ Redistributable：
   https://aka.ms/vs/17/release/vc_redist.x64.exe

Q: API 连接失败？
A: 检查「参数设置」页的 API Base URL 是否正确，
   确认 API Key 没有过期。

Q: 想自己改模型列表？
A: 「参数设置」页有模型管理入口，支持新增/删除。
   数据保存在 data/custom_models.json。
